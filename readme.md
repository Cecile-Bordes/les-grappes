# Test intégration Les grappes


## Installation

Utilisez le gestionnaire de paquets de votre choix.

```bash
npm install
```
or
```bash
yarn install
```

## Usage

Lancer le serveur de développement :

```bash
npm run dev
```
or
```bash
yarn dev
```
